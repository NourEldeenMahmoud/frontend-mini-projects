# Shortly URL Shortening Landing Page

A responsive landing page for a URL shortening service, built with Tailwind CSS.

## Features

- Responsive design
- Interactive link shortening form
- Custom color scheme
- Advanced statistics section with cards
- Mobile hamburger menu

## Usage

```bash
# Install dependencies
npm install

# Run Tailwind CLI in watch mode
npm run watch

# Build once
npm run build
```

## Live Preview

Open `index.html` in your browser after running the build command.
