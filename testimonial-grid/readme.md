# Testimonial Grid

A responsive testimonial grid layout built with Tailwind CSS, showcasing CSS Grid capabilities.

## Features

- CSS Grid layout with responsive breakpoints
- Testimonial cards with quotes, avatars, and status
- Color-coded cards for visual hierarchy

## Usage

```bash
# Install dependencies
npm install

# Run Tailwind CLI in watch mode
npm run watch

# Build once
npm run build
```

## Live Preview

Open `index.html` in your browser after running the build command.
