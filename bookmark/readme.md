# Bookmark Landing Page

A responsive bookmark manager landing page built with Tailwind CSS. Part of the Tailwind From Scratch course.

## Features

- Fully responsive design (mobile-first)
- Custom color scheme (soft blue, soft red)
- Interactive tabbed features section
- FAQ accordion
- Hamburger menu for mobile navigation

## Usage

```bash
# Install dependencies
npm install

# Run Tailwind CLI in watch mode
npm run watch

# Build once
npm run build
```

## Live Preview

Open `index.html` in your browser after running the build command.
