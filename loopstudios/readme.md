# Loopstudios Landing Page

A responsive landing page for a VR/immersive experiences company, built with Tailwind CSS.

## Features

- Responsive grid gallery of creations
- Mobile hamburger menu with overlay
- Hover effects on gallery items
- Custom fonts (Alata, Josefin Sans)

## Usage

```bash
# Install dependencies
npm install

# Run Tailwind CLI in watch mode
npm run watch

# Build once
npm run build
```

## Live Preview

Open `index.html` in your browser after running the build command.
