# Clipboard Landing Page

A responsive clipboard landing page built with Tailwind CSS. Based on the [Frontend Mentor Challenge](https://www.frontendmentor.io/challenges/clipboard-landing-page-5cc9bccd6c4c91111378ecb9).

## Features

- Responsive layout
- Custom color scheme (strong cyan, light blue)
- Download buttons for iOS and Android

## Usage

```bash
# Install dependencies
npm install

# Run Tailwind CLI in watch mode
npm run watch

# Build once
npm run build
```
