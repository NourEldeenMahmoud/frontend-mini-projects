# Fylo Landing Page (Dark Theme)

A responsive dark-themed landing page built with Tailwind CSS, featuring dark mode toggle support.

## Features

- Dark/light mode toggle
- Fully responsive (mobile-first)
- Custom dark theme color palette
- Feature cards, testimonial section
- Early access signup form
- Email validation

## Usage

```bash
# Install dependencies
npm install

# Run Tailwind CLI in watch mode
npm run watch

# Build once
npm run build
```

## Live Preview

Open `index.html` in your browser after running the build command.
